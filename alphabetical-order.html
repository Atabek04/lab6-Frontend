Exercise :

Write a function that returns a string in alphabetical order.
Example string: 'webmaster'
Expected Output: 'abeemrstw'

JavaScript Code:

function alphabet_order(str){
   return str.split('').sort().join('');
}
console.log(alphabet_order("webmaster"));
